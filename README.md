## Installation

```bash
$ npm install
```

## Setup Environment Variables
```bash
DB_USERNAME = postgres
DB_PASSWORD = 1234
DATABASE = todo_app
DB_HOST = localhost
DB_PORT = 5432
SALT = 10
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
